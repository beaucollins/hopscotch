Teleport

A task launcher for WordPress.

Installation

Download and install the extension for your browser:

  • Safari v1.0
  • Chrome v1.0

Usage

In any WordPress admin (version 3 or higher) double-press ⌘ to activate the Teleport interface.

It currently does 2 things:

  1. Indexes the navigation links in the left side navigation so you can type and hit enter to go to a new page.
  2. Provides search actions:
    • Search Posts:
    • Search Pages:
    • Search Media:
    • Search Comments:
    • Search Tags:
    • Search Categories:
